# AutoFlight
Model: Multiple Logistic Linear Regression<br>
# Run it with Terminal command:<br>
sudo python start.py checkpoints/gesture/ checkpoints/operation/
<br>
# parameters description:<br>
1."checkpoints/gesture/": gesture recognition checkpoint directory. (DeepLearning Model Hybrid-7 refer to <a href='https://github.com/mingweihe/HandGestureRecognition'>HandGestureRecognition</a>)<br>
2."checkpoints/operation/": autoflight checkpoint directory<br>
